## What is this?

This repo is the public-facing stats dashboard for TALON. Every hour, the TALON scheduler automatically pushes a fresh snapshot of the pipeline to this repo. The site auto-refreshes every hour.

## TALON System

TALON is an 18-phase autonomous AI assistant built by Thousifuddin Shaik:

- **Job Intelligence** — ingests 500+ jobs/day from 10+ sources, LLM scoring, resume tailoring
- **Gmail Monitoring** — OAuth2 parser detecting rejections, interviews, recruiter outreach
- **News Intelligence** — daily briefs on visa news, tech hiring, politics, world events
- **CRM** — tracks all applications with follow-up reminders
- **Multi-Model AI Gateway** — Gemini 3.1 → NVIDIA DeepSeek → OpenAI fallback chain

**Stack:** Node.js v20 · Ollama · Gemini 3.1 · GPT-4o · DeepSeek V3 · Discord API · Gmail OAuth2 · PM2 · n8n · Ubuntu 24 · KAMRUI Mini PC
